# FPS-Multijoueur-Raylib
## Description
Il s'agit d'un fps tous contre tous en multijoueur créé en C# avec la librairie Raylib

## Comment installer le jeux

- [ ] [Installer le zip](https://git.s2.rpn.ch/MonnierM/fps-multijoueur-raylib/-/archive/main/fps-multijoueur-raylib-main.zip)
- [ ] Décompresser le fichier
- [ ] Compiler le jeu (cela requiert le .net framework)

```
cd chemin/vers/mon/dossier
dotnet build
```

## Comment jouer
### Choix de l'option : Serveur ou Client
- Lorsque vous ouvrez l'application, vous devez choisir :
    - **Serveur** : Pour créer une partie que d'autres joueurs peuvent rejoindre.
    - **Client** : Pour rejoindre une partie déjà créée par quelqu'un d'autre.

### Si vous êtes le premier joueur
1. Lancez l'application **deux fois** sur votre PC :
    - Une fois pour sélectionner **Serveur** et créer la partie.
    - Une autre fois pour sélectionner **Client** et rejoindre votre propre serveur.

### Si vous voulez rejoindre la partie d'un autre joueur
1. Lancez l'application.
2. Choisissez **Client**.
3. Entrez l'**adresse IP** du PC de la personne qui a créé le serveur.
